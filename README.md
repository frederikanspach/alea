# ALEA — Secure Random Suite

ALEA ist ein moderner, minimalistischer und kryptografisch sicherer Zufallsgenerator. Entwickelt für Profis, die Wert auf Datensicherheit und sauberes Design legen.

## ✨ Features
- **Sicherheit geht vor:** Nutzt die native `window.crypto` API für echte Zufallszahlen.
- **Null Server-Kontakt:** Alle Berechnungen finden ausschließlich lokal in deinem Browser statt.
- **System-Optimiert:** Vordefinierte Zeichensätze für Linux, iOS, Windows und universelle Passwörter.
- **Batch-Generierung:** Erzeuge bis zu 99 Strings gleichzeitig mit individuellen Copy-Buttons.
- **User Experience:** Voller Dark Mode Support, persistente Einstellungen via LocalStorage und ein intuitives Interface.

## 🛠 Tech Stack
- **HTML5** (Semantische Struktur)
- **SCSS** (Modulares Styling & Dark Mode Management)
- **Vanilla JavaScript** (Leichtgewichtig & performant)
- **Vite** (Empfohlen als Build-Tool)

## 🚀 Installation & Deployment
Da ALEA eine reine Frontend-Anwendung ist, kann sie ohne Backend gehostet werden.

1. Repository klonen: `git clone https://github.com/DEIN-NAME/alea.git`
2. Lokal öffnen oder auf einen statischen Webserver (z.B. all-inkl.com oder GitHub Pages) hochladen.
