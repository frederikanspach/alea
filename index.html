Sicherheit & Datenschutz

ALEA nutzt die Web Crypto API für echte Zufallswerte. Die Generierung erfolgt ausschließlich lokal im Browser, es werden keine generierten Passwörter übertragen oder auf Servern gespeichert.

Lediglich die bevorzugten UI-Einstellungen (Länge, Anzahl und Design-Modus) werden lokal gespeichert, damit sie beim nächsten Besuch erhalten bleiben.
